# MovieMVP

Приложение на два экрана (скриншоты внизу)

Используемый стек:
- SwiftLint, SwiftFormatter
- Кеширование фотографий (Proxy)
- Работа с API TMDB
- AutoLayout, верстка кодом
- MVP 
- CoreData 
- Assembly
- UnitTests / UITests
- Router
- Alamofire
- SPM
- UITableView

Также есть этот же проект на MVVM с Realm, URLSession и Coordinator
